# User Service

User management microservice following OOP, Clean Code, MVC, and microservices architecture.

## Structure

- `src/models/` - Domain models (OOP classes)
- `src/services/` - Business logic (OOP classes)
- `src/repositories/` - Data access (OOP classes)
- `src/controllers/` - HTTP handlers (MVC)
- `src/routes/` - Route definitions
- `src/middleware/` - Express middleware

## Running

```bash
npm install
npm run dev
```

Service runs on port 8002 by default.
